# Learn Simple Database

## Deskripsi
Proyek sederhana CRUD dengan Golang dan MySQL

## Prasyarat
- Go 1.16+
- MySQL

## Instalasi
1. Clone repository
2. Impor dependensi
3. Atur koneksi database

## Penggunaan
- `go run main.go`
- `go test ./...`
